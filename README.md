# Proyecto Hardware EINA

## Descripción
Repositorio compartido para la asignatura de 3º de Informática de Proyecto Hardware en la Escuela de Ingeniería y Arquitectura (EINA).

## Description (English)
Shared repository for the 3rd year Computer Science Hardware Project course at the School of Engineering and Architecture (EINA).

## Estructura del Repositorio
Este repositorio está organizado para facilitar la colaboración entre estudiantes en los proyectos de hardware asignados durante el curso.

## Repository Structure
This repository is organized to facilitate collaboration among students on hardware projects assigned during the course.

## Uso
- Cada proyecto tendrá su propia carpeta
- Seguir las convenciones de nomenclatura establecidas
- Documentar adecuadamente todos los cambios

## Usage
- Each project will have its own folder
- Follow established naming conventions
- Properly document all changes

## Contribución
Para contribuir a este repositorio:
1. Crear una rama para tu trabajo
2. Realizar los cambios necesarios
3. Documentar los cambios realizados
4. Crear un pull request para revisión

## Contributing
To contribute to this repository:
1. Create a branch for your work
2. Make necessary changes
3. Document the changes made
4. Create a pull request for review
